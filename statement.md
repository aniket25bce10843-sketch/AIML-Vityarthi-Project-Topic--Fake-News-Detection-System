# Problem Statement
## This project aims to develop a Fake News Detection System that analyzes user- provided text and then classifies it as Real news or Fake news using machine learning technique. The goal is to provide simple tool whic will help to identify the news which is misleading news content.
---
# Scope Of The Project
## 1- The system mainly focuses on text based news classification.
## 2- A machine learning model is used for the prediction and the model is Logistic Regression.
## 3- The project includes a Graphical User Interface for the user interaction.
---
# Limitations:
## 1- It mainly works on a limited dataset.
---
# Target Users
## 1- Students learning Machine Learning can take help from this project
## 2- Educators or news media can use the fake nes detection system.
## 3-Normal users can check the news authencity by themselves.
---
# High-Level Features
## 1- The users can input any of the news text.
## 2- Simple GUI using TKinter.
## 3- Instant Result Display
