# AIML-Vityarthi-Project-Topic--Fake-News-Detection-System
## Fake News Detection System
# What is this project about?
## This project is a simple Fake News Detection System which verifies whether the news is real or fake. The project is built using python, machine learning and I have added GUI using Tkinnter. It tell whether the user- entered news text is Real or Fake. If the text is not entered by the user, it will show as enter the news text.
# Features
## 1- It detects whether the news is real or fake
## 2- Uses TF-IDF Vectorization for processing of the text
## 3- The project uses logistic regression for classification
## 4- It includes simple and interactive GUI.
# Technologies Used
## 1- Python
## 2- Scikit-learn 
## 3- TKinter
## 4- TF-IDF Vectorizer
